# あなたの作業チェックリスト(投資デイリー)

## 最優先(これでサイトが公開されます)

### 1. GitHubリポジトリ作成(2分)

1. https://github.com/new を開く
2. Repository name: `invest-daily` / **Public** を選択 → **Create repository**
3. 作成後、**Settings → Pages → Build and deployment → Source** を「**GitHub Actions**」に変更

→ ここまで終わればClaudeがpushして公開します。

### 2. SNSアカウントの方針を決める

投資デイリーのSNS告知用アカウントをどうするか、Claudeに伝えてください:

- **案A(推奨)**: 投資用に新しいアカウントを作る
  - Bluesky: 新規アカウント作成(5分)+アプリパスワード発行
  - Threads: 新しいInstagramアカウント+Metaアプリにテスター追加+トークン発行(AIデイリーでやった手順と同じ・約20分)
- **案B**: AIデイリーと同じアカウント(@ainews1 / @ainews111)に相乗りする(作業ゼロ、ただしテーマが混ざる)

決まったらClaudeに伝えてください。設定ファイル(.secrets.json)の準備はこちらでやります。

## 収益化(サイト公開後)

### 3. もしもアフィリエイト・A8.netに「サイト追加」

既にAIデイリーで登録済みのアカウントに、本サイト(https://9qu1.github.io/invest-daily/)を**2つ目のサイトとして追加**するだけでOKです。
投資系は「証券口座開設」案件が高単価(1件数千円〜)なので、AI系より収益ポテンシャルは高めです。
広告コードが発行されたらClaudeに渡してください → config/ads.json に設定します。

### 4. 独自ドメイン+AdSense(1〜2ヶ月後)

記事が溜まったらAIデイリーと同様に。1日3本ペースなら30本到達は約10日と早いですが、サイトの運用実績も見られるので**1ヶ月程度は待つのがおすすめ**です。
